Declaration: name and value separated by a colon.

CSS declaration: ends with semicolons, declaration blocks surrounded by curly braces.

Css Combinators

There are four different combinators in CSS3

  • descendant selector (space)
  • child selector(>)
  • adjacent sibling selector(+)
  • general sibling selector (~)

Descendant Selector: matches all elemtnas that are descendants of a specified element.

Child Selector: selects all elements that are the immediate children of a specified element.

Adjacent Sibling Selector: selects all elements that are the adjacent siblinsg of a specified element.

General Sibling Selector: selects all elements that are siblings of specified elements.
